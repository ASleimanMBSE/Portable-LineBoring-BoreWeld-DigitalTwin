Portable Line Boring & Bore Welding Digital Twin

This is a full-featured Digital Twin prototype for a portable in-field machining and restoration tool that performs both line boring and bore welding. This system is engineered for rugged environments, using open-source software, Arduino/RPi-based control, and real-time feedback sensors.

Architecture Diagram

System Highlights

  • Vibration, temperature, flow, and strain-based feedback
  • Live dashboarding using Grafana & Power BI
  • Tool wear prediction and bore welding process tuning
  • Built with Arduino, Raspberry Pi, and Node-RED
